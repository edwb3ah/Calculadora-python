# Calculadora em Python

Projeto de uma calculadora básica desenvolvida em Python.  

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Menu interativo para repetir operações
- Tratamento de divisão por zero

## Como usar
1. Abra o arquivo `calculadora.py` em um app Python (ex: Pydroid 3, Termux ou Replit)
2. Execute o código
3. Escolha a operação desejada e digite os números
4. Para sair, selecione a opção de saída no menu

## Objetivo do projeto
Este projeto tem como objetivo **demonstrar habilidades em lógica de programação**, estruturas condicionais, loops e manipulação de entrada do usuário.
